# Give it an input

*2026-07-02 · python-kata · progression track · +10 XP · `py-greet`*

## The problem

Functions can take input, called a parameter. Here `name` is handed to you. Return the text: Hello, <name>! — for example greet('Tyler') returns 'Hello, Tyler!'. Join pieces of text with the + sign.

## My solution

```python
def greet(name):
    # return "Hello, " + name + "!"
    return "Hello, " + name + "!"
```

## Verified against

- `greet('Tyler')` → `"Hello, Tyler!"` ✓
- `greet('world')` → `"Hello, world!"` ✓
